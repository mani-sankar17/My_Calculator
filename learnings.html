Working Mechanism:

  • First of all we have created the User Interface using HTML & CSS.
  • Defined same class name to the operands (i.e from 0-9) and operators (i.e -, +, *, /).
  • C (clear) and = (result) serve two different purposes, so both are defined with different class names.
  • Accessing the operands, operators, clear, result and input/display using DOM (Document Object Model).
  • Using addEventListener() function for updating the display based on the button clicks.
  • When clicked on ' = ' the eval() function processes the mathematical expressions and prints the result on the screen.
  • Using ' C ' we can clear the screen.
  • Included exception handling concepts (try, catch) to handle errors.
  • Achieved responsive web design using CSS media queries.
